# AI Agency Website Design Agent

## Objective
Design and build a premium, futuristic website for an **AI Agency** that develops AI systems, automation platforms, and intelligent digital products.

The website should feel **high-end, cinematic, and interactive**, similar to modern creative agencies such as:

- somstudios.com
- adko.in
- kyte-agency.com
- unlocking982.com

The motion and immersive feel should also be inspired by the **Lando Norris website**, which uses energetic transitions, bold typography, and dynamic interactions.

The goal is to create a **visually striking, modern agency site that communicates innovation, speed, and technical sophistication.**

---

# Design Philosophy

The website should combine three core themes:

1. **AI Intelligence**
2. **Creative Agency Design**
3. **Formula-1 Speed and Energy**

The experience should feel **fast, futuristic, and premium.**

---

# Visual Style

Use a **dark premium interface**.

Primary colors
- Black
- Charcoal
- Deep graphite

Accent colors
- Electric green
- Neon purple
- Gradient orange

Design elements
- glassmorphism panels
- glowing gradients
- animated particles
- digital grid backgrounds
- neural network visuals

Typography
- large bold display fonts
- clean modern sans-serif
- strong editorial hero headlines

---

# Core UX Principles

The website should feel **interactive and immersive**.

Include:

- smooth scrolling
- parallax sections
- hover distortions
- animated cards
- cursor interactions
- motion-based transitions
- subtle WebGL effects

Scrolling should feel **fast and fluid**, inspired by racing telemetry and Formula 1 dashboards.

---

# Page Structure

## 1. Hero Section

Full screen immersive hero.

Background:
- animated neural network
- subtle moving grid
- particle animation

Headline:
"Building the Future with AI"

Subheadline:
"We design AI systems, automation tools, and intelligent digital products that scale businesses."

Buttons:
- View Work
- Start a Project

Include subtle cursor interaction and a scroll indicator.

---

## 2. Services Section

Display services using animated cards.

Services include:

- AI Product Development
- AI Automation
- AI Agents
- LLM Integration
- Data Intelligence
- AI SaaS Platforms

Cards should glow and expand slightly on hover.

---

## 3. Case Studies / Projects

Showcase selected work.

Each card should contain:

- project image
- hover video preview
- short description
- tech stack

Clicking opens a detailed project page.

---

## 4. AI Capabilities

Display capabilities as an animated grid.

Examples:

- autonomous AI agents
- generative AI systems
- workflow automation
- predictive analytics
- AI copilots
- knowledge systems

Icons animate when they appear in the viewport.

---

## 5. Process Section

Show a 5-step workflow.

Steps:

1. Discover
2. Architect
3. Build
4. Train
5. Deploy

Design this section like a **racing track timeline**.

---

## 6. Social Proof

Display client logos and testimonials.

Testimonials should appear in a **horizontal scrolling carousel.**

---

## 7. Team Section

Editorial style presentation of team members.

Headline:
"AI Engineers. Product Thinkers. Builders."

Large cinematic portraits.

Minimal descriptions.

---

## 8. Call To Action

Large high-impact section.

Headline:
"Let's Build Your AI System"

Buttons:

- Book a Call
- Start a Project

Background animation should intensify here.

---

## 9. Footer

Minimal dark footer.

Sections:

- Company
- Services
- Work
- Blog
- Contact

Include social links.

---

# Interaction and Motion

Animations should include:

- smooth section transitions
- subtle parallax effects
- hover glow effects
- animated gradients
- card tilt interactions
- page transition animations

All animations should feel **fast, responsive, and premium.**

---

# Technology Stack

Assume implementation with:

- Next.js
- TailwindCSS
- Framer Motion
- GSAP
- Three.js
- Lenis smooth scroll

The code should be modular and production ready.

---

# Performance Requirements

- fast load times
- optimized animations
- responsive design
- mobile-friendly interactions
- accessibility compliant

---

Produce a **world-class AI agency website** that could compete with **top design studios and award-winning agency sites.**

The final design should feel:

- futuristic
- premium
- bold
- fast
- intelligent
